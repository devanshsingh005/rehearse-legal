Basic Troubleshooting

  • Make sure your app is updated to the latest available version.
  • Check that your device is running a supported version of iOS.
  • Try closing and reopening the app if a screen becomes unresponsive.
  • For upload issues, confirm the file is clear, complete, and readable before retrying.
